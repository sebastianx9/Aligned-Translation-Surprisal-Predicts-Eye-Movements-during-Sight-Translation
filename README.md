# Aligned Translation Surprisal Predicts Source-Text Eye Movements During Sight Translation

This repository contains the code for extracting aligned translation surprisal and reproducing the mixed-effects model and cross-validation analyses reported in the paper.

## Data

Eye-tracking data are from the **EMMT corpus** (Bhattacharya et al., 2022), available at:
https://github.com/rgu-iit-bt/emmt

Data files are not included in this repository. Place the following files in the `data/` folder before running the analysis:

| File | Description |
|------|-------------|
| `fixation_durations_word.csv` | Word-level total fixation duration (output of `src/extract_fixation_duration.py`) |
| `nmt_surprisal_soft_word.csv` | Aligned translation surprisal (output of `src/extract_nmt_surprisal_soft.py`) |
| `monolingual_surprisal_word.csv` | GPT-2 monolingual surprisal (output of `src/extract_monolingual_surprisal.py`) |
| `subtlex_us.csv` | SUBTLEX-US word frequencies (Brysbaert & New, 2009) |

## Repository Structure

```
├── src/
│   ├── extract_fixation_duration.py      # Raw gaze data → word-level TFD
│   ├── extract_nmt_surprisal_soft.py     # NMT decoder surprisal with soft alignment
│   ├── extract_monolingual_surprisal.py  # GPT-2 monolingual surprisal
│   └── plot_alignment.py                 # Cross-attention alignment visualisation
├── data/                                 # Place preprocessed CSV files here
├── figures/                              # Generated by result-analysis.ipynb
├── result-analysis.ipynb                 # Main analysis (R kernel)
└── requirements.txt                      # Python dependencies
```

## How to Run

**Step 1: Extract features** (requires EMMT corpus and Python dependencies)

```bash
pip install -r requirements.txt
python src/extract_fixation_duration.py
python src/extract_nmt_surprisal_soft.py
python src/extract_monolingual_surprisal.py
```

**Step 2: Run the analysis** (requires R with IRkernel)

```bash
# Install R dependencies (once)
Rscript -e 'install.packages(c("lme4", "lmerTest", "dplyr", "ggplot2"))'

# Open and run the notebook
jupyter notebook result-analysis.ipynb
```
